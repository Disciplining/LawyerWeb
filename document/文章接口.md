#### <font color="blue">所有文章类型下详情接口</font>
- 请求地址：`/allTypeDetail?id=xx`
- 请求方式：`GET`
- 返回数据类型：`json`
- 返回数据示例
    ```json
    {
        "idArr": [64,65,71,72,73,74,75,76,77,78,79],
        "articleDetail": 
        {
            "articleId": 77,
            "type": 2,
            "articleTitle": "这是一个文章的名字-32",
            "author": "123-f",
            "intro": "11111",
            "content": "aaasdasd",
            "editDate": "2019-08-14",
            "count": 0,
            "deleteFlag": "0",
            "articleImg": null
        }
    }
    ```

#### <font color="blue">所有类型的文章分页接口</font>
- 请求地址：`/allArticlePageData?pageNum=xx&pageSize=xx`
  + pageNum：当前页码，从1开始计
  + pageSize：每页数据的条数
- 请求方式：`GET`
- 返回数据类型：`json`
- 返回数据示例
    ```json
    {
        "code": 0,
        "data": [
            {
                "articleId": 67,
                "type": 3,
                "articleTitle": "文章名字",
    ```
